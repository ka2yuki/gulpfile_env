# Scssメリット
- ネストして書けるので、スクロールを省くことができる.
- ネスト（入れ子）が使えて構造が把握しやすい
- 自動でコメントを削除してくれる.
- ファイルを分割できる.
  - スクロールを減らせる.
- ~変数が使える(cssでも使える）~
- Mixin（ミックスイン）でコードを引用できる.
- コードをincludeして使い回せる.（継承）
- 関数が使える. (mixinと違いがよく分からない)
- [more details..](https://digitalidentity.co.jp/blog/creative/css-sass.html)
