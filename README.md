# Scssメリット
- 自動でコメントを削除してくれる.
- ファイルを分割できる（関心の分離）
  - スクロールを減らせる.
- ネスト（入れ子）が使えて構造が把握しやすい
- 変数が使える
- Mixin（ミックスイン）でコードを引用できる
- コードを継承で使い回せる
- 関数が使える
